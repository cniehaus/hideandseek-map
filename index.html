About this tool ▾

Jet Lag Maps is a companion tool for the board game Jet Lag: The Game – Hide & Seek, inspired by the YouTube channel Jet Lag: The Game.

The tool shows interactive maps with postal code districts, train stations, hospitals, landmarks, stadiums and more for any city – print-ready in A4 format. All map data comes from OpenStreetMap.
